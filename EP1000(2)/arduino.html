Arduino


We learned what electricity is and how electrical components work. We also learned how to utilize a breadboard, which serves as a foundation for developing electronic circuits. It's utilized in a lot of prototyping projects, and it's great for quick circuit testing. We were given the challenge of simulating a 555 timer in TinkerCAD. The 555 timer is set up as an astable with R and C values, producing an alternating ON/OFF signal that flashes the LED.

Here is my simulated 555 timer in TinkerCAD In TinkerCAD, I created a mimicked 555 timer. Embedded systems were also covered. A dedicated computer system having a dedicated function within a larger system to perform a specific task is known as an embedded system. Smart TVs and stoves, for example. We'll be using Arduino Systems for this project. It includes open source libraries as well as support. Because there are so many clones and third-party makers, it is affordable. It has a cross-platform development environment with a basic, straightforward GUI. Next, I learned how to program with Arduino. The Arduino code is loosely based on C ++. The digital I / O pin can input or output a digital (0.5V) signal. The Arduino system provides three functions for manipulating digital I / O. I also received an explanation about analog I / O. Analog signals must be converted to digital values ​​before they can be processed.


The Code:


// Processing code // #define SW 2 #define GRE 13 #define RED 12 #define YEL 11 int count = 0; // Initial count value int state = HIGH; // Button not pressed void setup() { Serial.begin(9600); Serial.println("Start"); pinMode(SW, INPUT); pinMode(GRE, OUTPUT); pinMode(RED, OUTPUT); pinMode(YEL, OUTPUT); decode(count); } void loop() { if(digitalRead(SW) == LOW){ // Switch is pressed if (state == HIGH){ // Process state = LOW; } else { // do nothing } } else { if (state == LOW){ // Registers the keypress count = (count + 1)%6; decode(count); state = HIGH; } } } // Decodes count into binary void decode(int v) { switch(v){ case 0: // LEDs all off digitalWrite(RED, 0); digitalWrite(GRE, 0); digitalWrite(YEL, 0); break; case 1: // Only red LED on digitalWrite(RED, 1); digitalWrite(GRE, 0); digitalWrite(YEL, 0); break; case 2: // Only green LED on digitalWrite(RED, 0); digitalWrite(GRE, 1); digitalWrite(YEL, 0); break; case 3: // Only yellow LED on digitalWrite(RED, 0); digitalWrite(GRE, 0); digitalWrite(YEL, 1); break; case 4: // All LEDs on digitalWrite(RED, 1); digitalWrite(GRE, 1); digitalWrite(YEL, 1); break; case 5: // LEDs flash green then red then yellow digitalWrite(RED, 1); digitalWrite(GRE, 0); digitalWrite(YEL, 0); delay (500); digitalWrite(RED, 0); digitalWrite(GRE, 1); digitalWrite(YEL, 0); delay (500); digitalWrite(RED, 0); digitalWrite(GRE, 0); digitalWrite(YEL, 1); delay (500); break; } }
